How does it works ?

All the music notes are saved into a database, which allows us to play the tune. Every notes are connected to a finger.
Depending on which song you have selected, a partition will play, which has previously been coded on Arduino.

Other uses:

InMoov Hand is a robotic hand which can use in many situations.
For example, it can be use as a hand prosthesis for disabled people.

Or it can be for any entertaining purposes.

This experience will may be a success for the future.
